⇌ FastAPI Integration Setup Guide
1. Install ChromaDB
# Install via pip pip install chromadb sentence-transformers
2. Start ChromaDB Server (HTTP mode)
# Run as a standalone HTTP server chroma run --path ./chroma_data --host 0.0.0.0 --port 8001 # Or with Docker docker run -p 8001:8000 -v ./chroma_data:/chroma/chroma chromadb/chroma
3. Connect this UI to your server
# Click "Connect" in the top right and enter: Host: localhost Port: 8001 # (or your deployed server URL)
4. FastAPI — chroma_client.py
import chromadb from chromadb.config import Settings # HTTP client (connects to running server) client = chromadb.HttpClient( host="localhost", port=8001, settings=Settings(anonymized_telemetry=False) ) # Application Collections resume_coll = client.get_or_create_collection( name="resume_chunks", metadata={"hnsw:space": "cosine"} ) jd_coll = client.get_or_create_collection( name="job_descriptions", metadata={"hnsw:space": "cosine"} ) interview_coll = client.get_or_create_collection( name="interview_qa", metadata={"hnsw:space": "cosine"} )
5. RAG Retrieval in your FastAPI route
from sentence_transformers import SentenceTransformer from fastapi import FastAPI app = FastAPI() embedder = SentenceTransformer("all-MiniLM-L6-v2") @app.post("/analyze") async def analyze_resume(query: str, n_results: int = 5): query_embed = embedder.encode([query]).tolist() results = resume_coll.query( query_embeddings=query_embed, n_results=n_results, include=["documents", "metadatas", "distances"] ) return { "chunks": results["documents"][0], "metadata": results["metadatas"][0], "scores": results["distances"][0] }
6. Add CORS for this UI (FastAPI)
from fastapi.middleware.cors import CORSMiddleware app.add_middleware( CORSMiddleware, allow_origins=["*"], # or specific origin allow_methods=["*"], allow_headers=["*"], )
💡 Pro tip for hybrid search
Since you're using FAISS+BM25 hybrid retrieval, ChromaDB works best as a persistent store for your chunks. Store chunks once → query with ChromaDB → re-rank with your CrossEncoder → pass to LLM. This gives you a visual admin UI (this page) without changing your retrieval pipeline.
